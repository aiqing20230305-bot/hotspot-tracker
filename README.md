## 🚀 快速开始

### 方式1：交互式启动（推荐）

```bash
python3 start.py
```

### 方式2：命令行模式

```bash
# 运行完整工作流
python3 automation_workflow.py --run

# 查看系统状态
python3 automation_workflow.py --status

# 生成优化报告
python3 self_learning.py --report
```

### 方式3：定时自动执行

```bash
# 设置定时任务
python3 automation_workflow.py --setup-cron

# 手动添加到crontab
crontab -e
# 添加: 0 9 * * * cd /path/to/hotspot-tracker && python3 start.py
```

## 📱 核心功能

### 1️⃣ 智能热点抓取

- **抖音**：实时热搜榜（50条）
- **微博**：实时热搜榜（50条）
- **小红书**：热搜榜+垂直搜索（20条）
- **智能匹配**：自动关联9大行业

### 2️⃣ AI内容推荐

- **标题生成**：基于热点+品牌自动生成
- **平台分配**：根据历史效果智能分配
- **角度选择**：产品测评/教程/场景植入等
- **CTA优化**：自动生成互动引导语

### 3️⃣ 效果反馈学习

```python
# 添加内容效果反馈
python3 self_learning.py --add-feedback '{
    "content_id": "content_001",
    "industry": "3C数码",
    "platform": "抖音",
    "angle": "产品测评",
    "hot_topic": "华为千元新机",
    "metrics": {
        "likes": 15000,
        "comments": 800,
        "shares": 300,
        "views": 200000
    }
}'
```

### 4️⃣ 自学习优化

系统自动分析：
- ✅ 行业表现排名
- ✅ 平台效果对比
- ✅ 内容角度效果
- ✅ 热点借势成功率
- ✅ 最佳组合推荐

## ⚙️ 配置说明

### 飞书推送配置

```bash
python3 automation_workflow.py --config-feishu "https://open.feishu.cn/open-apis/bot/v2/hook/xxx"
```

### 钉钉推送配置

```bash
python3 automation_workflow.py --config-dingtalk "https://oapi.dingtalk.com/robot/send?access_token=xxx"
```

### 定时任务配置

```bash
# 每天9:00抓取热点，9:30生成报告，10:00推送
0 9 * * * cd /path/to/hotspot-tracker && python3 evolution_system.py
30 9 * * * cd /path/to/hotspot-tracker && python3 automation_workflow.py --run
0 10 * * * cd /path/to/hotspot-tracker && python3 automation_workflow.py --run
```

2. **9:30** 生成内容推荐报告
3. **10:00** 推送到飞书/钉钉
4. **内容发布后** 及时添加效果反馈
5. **每周一** 查看自学习优化报告
6. **持续迭代** 系统会自动优化推荐算法
