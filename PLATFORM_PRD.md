# 🎯 特赞内容运营平台 - 产品需求文档

## 核心定位
专属于特赞的内容运营智能平台，从热点追踪到内容生产的一站式解决方案

---

## 功能模块

### 1️⃣ 热点追踪中心
- [x] 每日热点TOP10（抖音/微博/小红书）
- [x] 热点背后逻辑分析
- [x] 热度趋势预测
- [ ] 热点日历（提前预知节日/大促热点）

### 2️⃣ 低粉爆款榜 ⭐ NEW
**价值**：低粉账号的爆款内容更值得借鉴（非粉丝效应，纯内容驱动）

**数据维度**：
- 粉丝数 < 1万，点赞 > 10万
- 粉丝数 < 5万，点赞 > 50万
- 粉丝数 < 10万，点赞 > 100万

**展示字段**：
- 账号信息（昵称/粉丝数/领域）
- 爆款内容（标题/点赞/评论/转发）
- 内容标签/话题
- 发布时间
- 可复制的方法论提炼

**时间维度**：
- 日榜：昨日低粉爆款
- 周榜：本周低粉爆款TOP20
- 月榜：本月低粉爆款TOP50

### 3️⃣ 客户选题中心 ⭐ NEW
**按行业维度**：
- 3C数码（荣耀/罗技）
- 快消（AHC/多芬/力士/清扬/舒适）
- 家庭清洁（HC）
- 保健品（汤臣倍健）
- ...12个行业

**每个客户展示**：
- 客户基础信息（品牌/SKU数/优先级）
- 可结合的热点（智能匹配）
- 推荐选题（3-5个/日）
- 历史内容效果数据

### 4️⃣ SKU场景库 ⭐ NEW
**每个SKU包含**：
- 产品基础信息（名称/规格/卖点）
- 使用场景（5-10个）
- 用户痛点（3-5个）
- 情感共鸣点（3-5个）
- 内容角度建议（测评/教程/场景植入）

**场景示例**（AHC水乳）：
- 场景1：春日妆容打底 → 痛点：卡粉脱妆 → 共鸣：想要精致妆容
- 场景2：熬夜后急救 → 痛点：暗沉干燥 → 共鸣：拯救熬夜肌
- 场景3：学生党护肤 → 痛点：预算有限 → 共鸣：平价好物

## 数据库设计

### 低粉爆款表
```sql
CREATE TABLE low_fan_hits (
    id INT PRIMARY KEY,
    platform VARCHAR(20),      -- 平台：抖音/小红书/B站
    account_name VARCHAR(100), -- 账号名
    fan_count INT,             -- 粉丝数
    content_title TEXT,        -- 内容标题
    content_url VARCHAR(500),  -- 内容链接
    like_count INT,            -- 点赞数
    comment_count INT,         -- 评论数
    share_count INT,           -- 分享数
    publish_date DATE,         -- 发布日期
    category VARCHAR(50),      -- 分类
    tags TEXT,                 -- 标签
    methodology TEXT,          -- 方法论提炼
    created_at TIMESTAMP
);
```

### SKU场景表
```sql
CREATE TABLE sku_scenarios (
    id INT PRIMARY KEY,
    client_id INT,             -- 客户ID
    sku_name VARCHAR(100),     -- SKU名称
    sku_specs TEXT,            -- 规格参数
    core_selling_points TEXT,  -- 核心卖点
    scenario_name VARCHAR(100),-- 场景名称
    user_pain_point TEXT,      -- 用户痛点
    emotional_resonance TEXT,  -- 情感共鸣
    content_angles TEXT,       -- 内容角度建议
    hot_topics_match TEXT      -- 可匹配热点
);
```
